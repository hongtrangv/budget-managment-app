# Product Overview

This is a library management system (Trình quản lý nội bộ) with financial tracking capabilities. The application provides:

- **Book Management**: Catalog books with shelf location tracking and visual library layout
- **Collections Management**: Organize books by categories and genres
- **Loan Payment System**: Track book loans and financial transactions
- **Dashboard & Reporting**: Analytics and insights on library operations
- **AI Chatbot Assistant**: Financial assistant for user queries (powered by OpenAI)

The system is designed for internal use with Vietnamese language support throughout the UI.

## Key Features

- Visual shelf layout system with configurable rows and compartments
- Real-time data synchronization with Firebase/Firestore
- Single-page application (SPA) architecture for smooth navigation
- Redis caching for performance optimization
- API key authentication with action-based authorization
- Responsive design with mobile menu support
